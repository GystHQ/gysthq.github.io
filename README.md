# gysthq.github.io

Landing page for **thomasanakin.com** — the GYST Games hub.

This is the GitHub Pages **user/organization site** for `GystHQ`. Because the
apex domain `thomasanakin.com` is attached to *this* repo, every other GystHQ
repo with Pages enabled is served at `thomasanakin.com/<repo-name>/`:

| Path | Repo | Game |
|---|---|---|
| `/` | `gysthq.github.io` (this repo) | Landing page |
| `/worldcup/` | `worldcup` | World Cup 2026 Cards |
| `/f1/` | `f1` | F1 — Apex Collectors |

Static site — no build step. Edit `index.html` and push; Pages serves the
`main` branch root. The custom domain is configured in **Settings → Pages**
(do not commit a `CNAME` file unless the domain lives here).
